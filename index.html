NOTE

The number in the circle represents the number of hospitals in the area and NOT the number of beds.

To get bed availability details, click on the blue marker.

We use government data to get the hospital status and numbers. The actual number of availability of beds may differ. Please contact the hospital prior visiting.

The list of all data sources is available here. To report issues, please reach out us on GitHub or Twitter.

NOTE
This does not show availability of ICU beds, it only lists hospitals with ICU beds facility.

West Bengal releases no information about the total number of ICU beds in a hospital.
NOTE
This does not show availability of Ventilator beds, it only lists hospitals with Ventilator beds facility.

Madhya Pradesh, Telangana and West Bengal release no information about the total number of Ventilator beds in a hospital.
